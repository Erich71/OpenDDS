OpenDDS - Performance Testing Environments

OCILab

The test environment includes the test hosts and the network equipment connecting the hosts. All hosts in the tests were the same and a single network was used to connect them.
  1. Test Hosts
    • HP D530,P4HT
    • Intel(R) Pentium(R) 4 CPU 2.60GHz
    • 512K cache
    • 1.5 GB RAM
    • Tigon3 [partno(BCM95782A50) rev 3003, PHY(5705)] (PCI:33MHz:32-bit), 10/100/1000Base-T Ethernet
    • OpenSuse 9.1, Linux kernel 2.6.27.7-9-pae SMP
  2. Test Network
    • HP ProCurve switch model 2824
The network performance was characterized to determine the maximum bandwidth available for testing. The test environment can only support 596 Mb/s or just under 60% of the rated gigahertz bandwidth.
